# pytest-jupyter-deploy

Pytest plugin for E2E testing of jupyter-deploy templates.

## Overview

This package provides pytest fixtures for writing end-to-end tests for jupyter-deploy templates.

## Installation

```bash
pip install pytest-jupyter-deploy
```

## Usage

The plugin is automatically loaded by pytest when installed. Simply install the package and use the provided fixtures in your tests.
